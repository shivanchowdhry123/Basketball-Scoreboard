# Basketball Scoreboard

## JavaScript Fundamentals: Basketball Scoreboard

A functional, dark-themed basketball scoreboard application constructed during introductory JavaScript training to practice multiple independent state counters, interactive increments, and layout design.

### Features

*   **Dual Team Tracking:** Dynamically tracks points independently for both "Home" and "Guest" teams.
*   **Multi-Point Increments:** Implements standard basketball scoring options via responsive button interfaces ($+1$, $+2$, and $+3$ point controls).
*   **Foul & Period Tracking:** Features localized interactive controls to log active fouls per team and cycle through game periods.
*   **State Reset Utility:** Includes a "New Game" operational control to wipe current metrics and restore all counters back to zero.

## Tech Stack

*   **HTML5:** Structured semantic markup, application containers, and button controls.
*   **CSS3:** Dark-mode presentation layers, box-shadow depth layouts, glowing typography displays, and unified block spacing.
*   **JavaScript (ES6):** State modification logic, functional data updates, and dynamic DOM text bindings.

## Setup and Installation

1. Clone or download this project repository.
2. Open `index.html` within any modern web browser to view and interact with the scoreboard application.
